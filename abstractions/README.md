Managed types for the [vīcĭus updater agent](https://github.com/nefarius/vicius) JSON payloads.

> **Note:** This library is co-located in the [vicius](https://github.com/nefarius/vicius) monorepo so that model
> changes in the C++ updater agent and the C# abstractions can be kept in sync in a single pull request.
> The previous standalone repository ([Nefarius.Vicius.Abstractions](https://github.com/nefarius/Nefarius.Vicius.Abstractions))
> has been archived; please open issues and PRs here instead.
